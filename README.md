# Woice Landing Page

Initial landing page setup for Woice, a testimonial and customer feedback platform for small businesses.

## Stack

- Next.js App Router
- JavaScript only
- Tailwind CSS
- shadcn/ui

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Project Notes

This repository currently contains only the landing page foundation:

- App Router structure under `src/app`
- shadcn/ui configured for JavaScript
- clean landing section component structure
- basic SEO metadata
- responsive placeholder sections with a more polished hero
